# Umi-Log

Lightweight, easy-to-use and customizable console logger.

**Features**

* Timestamp 

* Colours

* Lightweight

* Customizable

**Installation**

``npm install umi-log``

**Usage**
```js
const log = require("umi-log");

log.info("Test");
log.warn("Test");
log.error("Test");
log.debug("Test");
log.custom("TEST", "Test");
```
